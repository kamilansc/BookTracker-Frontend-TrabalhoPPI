# BookTracker — Front-end

## Sobre o projeto

O BookTracker é uma aplicação web desenvolvida para acompanhar leituras e sessões de leitura de livros.

A aplicação permite cadastrar livros, registrar sessões realizadas, acompanhar progresso de leitura e visualizar informações organizadas em uma interface simples e intuitiva.

Este repositório contém exclusivamente o desenvolvimento do front-end da aplicação.



## Funcionalidades principais

* Adicionar livros em andamento
* Registro e acompanhamento das sessões de leitura
* Exibição do progresso de leitura por livro
* Visualização da última sessão de leitura realizada
* Consulta de detalhes do livro

* **components/** → componentes reutilizáveis da interface
